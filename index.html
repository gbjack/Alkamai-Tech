Join API evangelist, Kirsten Hunter for an in-depth, hands-on session surrounding API development on the platform

You will learn how to leverage the OPEN APIs to integrate with your systems or monitor your Akamai properties.

This workshop includes:

  • Participants will use provided application code to access Akamai APIs
  • Tour of Akamai's documentation portal
  • In depth presentation on Akamai API Authentication
  • Overview of API administration and provisioning tools
  • Deep dive into API Catalog and resources
  • Instructor-led exercises to demonstrate code creation, testing and troubleshooting
  • A Github repository will be used for demonstrations and exercises
  • Use cases of different APIs
